API Reference
GET / Health check + API docs
Response
{ "name": "TextShield API", "version": "1.0.0", "endpoints": { ... } }
POST /humanize Humanize AI text
Request body
{
  "text": "string (required) — the AI-generated text to humanize"
}
Response
{
  "success": true,
  "original": "string",
  "humanizedText": "string",
  "changesCount": 42,
  "changeRate": "18%",
  "wordCount": 230
}
cURL example
curl -X POST http://localhost:3000/humanize \
  -H "Content-Type: application/json" \
  -d '{"text": "Utilize this tool to facilitate better outcomes."}'
POST /detect Detect AI-generated text
Request body
{
  "text": "string (required) — minimum ~20 words for reliable results"
}
Response
{
  "success": true,
  "aiProbability": 87,
  "humanProbability": 13,
  "verdict": "AI Generated",
  "confidence": "High",
  "signals": {
    "aiKeywordDensity": 72,
    "sentenceUniformity": 65,
    "passiveVoice": 40,
    "transitionWordOveruse": 80,
    "structureRepetition": 30,
    "formalPhraseUsage": 60
  },
  "detectedKeywords": ["utilize", "facilitate", "furthermore"],
  "wordCount": 150,
  "sentenceCount": 8,
  "avgWordsPerSentence": 19
}
POST /plagiarism Check for plagiarism
Request body
{
  "text": "string (required)",
  "compareTexts": ["string", "string"]  // optional array of texts to compare
}
Response
{
  "success": true,
  "originalityScore": 92,
  "plagiarismScore": 8,
  "verdict": "Original",
  "matches": [
    {
      "source": "Document 1",
      "similarity": 8,
      "matchedPhrases": ["...machine learning algorithms..."]
    }
  ],
  "analysis": {
    "uniqueTrigramCount": 340,
    "totalSentences": 12,
    "avgSentenceLength": 18
  }
}
POST /analyze Run all three checks at once
Request body
{
  "text": "string (required)",
  "compareTexts": ["string"]  // optional
}
Response
{
  "success": true,
  "humanize": { ... },
  "detect": { ... },
  "plagiarism": { ... }
}
Quick Start
1. Start the server
node server.js
2. Test it
curl http://localhost:3000/
3. Humanize some text
curl -X POST http://localhost:3000/humanize \
  -H "Content-Type: application/json" \
  -d '{"text": "Furthermore, it is important to note that utilizing this comprehensive framework will significantly enhance overall performance."}'
